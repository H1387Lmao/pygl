# pygl
Python Graphics Library
### Info
PYGL or Python Graphics Library not to be confused by [PYGL](https://github.com/rajeshrinet/pygl) by rejeshrinet
is a python wrapper for pygame.

## Features
|Feature|Implemented|
|-------|---------------|
|Squares|    Yes        |
|Sprites|    Yes        |
|Circles|    No         |
|Input  | *Keyboard Only*|
